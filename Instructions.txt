✅ Comment l’utiliser :

Sauvegarde ton projet React + Node WebSocket dans /home/ubuntu/anaveo-project.

Place ce script deploy_anaveo.sh dans /home/ubuntu/.

Rends-le exécutable :

chmod +x /home/ubuntu/deploy_anaveo.sh


Lance le déploiement :

/home/ubuntu/deploy_anaveo.sh


Le script va :

Installer Node.js, Nginx, Certbot si nécessaire

Construire React

Déployer Node WebSocket via PM2

Configurer Nginx pour React + WebSocket avec HTTPS

Vérifier que le WebSocket fonctionne avant de publier la nouvelle build

Garder des backups et effectuer un rollback en cas de problème
